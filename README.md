# Poppang — Next.js + Tailwind Starter

간단한 Next.js + TypeScript + Tailwind 템플릿입니다. 홈(히어로/폰 목업)과 블로그 목록 페이지를 포함합니다.

빠른 시작

```bash
# 의존성 설치
npm install

# 개발 서버
npm run dev
```

블로그 포스트는 `posts/*.md`로 작성하며, frontmatter에 `title`, `date`, `category`, `excerpt`, `cover`를 추가하세요.

배포: Vercel 또는 Netlify 권장
